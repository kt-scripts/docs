KT Extra Menu is a lightweight FiveM script by **Lation Scripts** that gives players a convenient interface to select vehicle liveries (paint jobs) and toggle vehicle extras (e.g. spoilers, additional lights, light bars). The script supports several popular frameworks, two different menu systems, and can be fully customized through a clean `config.lua`.

## Features

<CardGroup cols={2}>
  <Card title="Multi-Framework Support" icon="layer-group">
    Natively compatible with **ESX**, **QBCore**, **QBox** and **Standalone** – no source code modifications required.
  </Card>

  <Card title="Two Menu Systems" icon="window">
    Choose between the widely used **ox\_lib** menu and the exclusive **Lation Modern UI** for a modern, branded look.
  </Card>

  <Card title="Radial Menu Integration" icon="circle-dot">
    Optional integration with a **Radial Menu** – players can open the menu directly from the radial menu without typing a command.
  </Card>

  <Card title="Custom Livery Names" icon="tag">
    Give each livery a custom, readable name in the configuration instead of only showing numeric IDs.
  </Card>

  <Card title="Job Restrictions" icon="briefcase">
    Restrict access to specific **jobs** (ESX/QBCore) so that, for example, only police officers or firefighters can switch certain liveries.
  </Card>

  <Card title="Multilingual (DE / EN)" icon="language">
    Bundled locale files for **German** and **English** – easily extendable to additional languages.
  </Card>

  <Card title="Version Check" icon="shield-check">
    Automatic **version check** on server start informs admins about available updates without interrupting gameplay.
  </Card>

  <Card title="Lua Exports" icon="code">
    Open the menu programmatically from other scripts via the provided **Lua exports** (`OpenVehicleMenu`, `OpenLiveryMenu`, `OpenExtrasMenu`) – ideal for integrations into your own resources.
  </Card>
</CardGroup>

## Dependencies

KT Extra Menu requires at least one of the following resources, depending on your menu choice:

| Resource | Type | Description |
| --- | --- | --- |
| [ox\_lib](https://github.com/overextended/ox_lib) | **Required** | Base library for menus, notifications and other helper functions. Must be started before `kt_extra_menu`. |
| [Lation Scripts Modern UI](https://discord.gg/A6UgrEhc8G) | Optional | Alternative, modern menu interface by Lation Scripts. Only required when `Config.MenuSystem = 'lation'` is set. |

<Note>
  `ox_lib` is always required, even when using the Lation Modern UI as the menu system, because the script relies internally on ox\_lib functions.
</Note>
