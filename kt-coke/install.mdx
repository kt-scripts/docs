<Info>
  Requires **ox\_lib**, **oxmysql**, and an inventory system.
</Info>

## 1. Download

Download `kt_coke` from [keymaster](https://keymaster.fivem.net) and extract to `resources/[kt]/kt_coke`.

## 2. Database

```bash
mysql -u root -p your_db < resources/[kt]/kt_coke/kt_coke.sql
```

## 3. Configuration

```lua config.lua
Config = {}
Config.Framework  = 'auto'
Config.Inventory  = 'ox'
Config.CartelRaids = true
Config.SellRoutes = { 'vagos', 'ballas', 'families' }
```
