KT Extra Menu can be opened in multiple ways: via chat command, via keybind or through a radial menu. All methods are centrally configurable in `config.lua` — without touching a single line of script code.

## Chat commands

| Command | Description |
| --- | --- |
| `/vehiclemenu` | Opens the vehicle customization menu (liveries & extras) |

The default command name is `vehiclemenu`. It can be changed at any time via `Config.Command` in `config.lua`:

```lua
Config.Command = 'vehiclemenu' -- enter any name
```

<Note>
  The command is always active, regardless of whether a keybind or the radial menu is enabled.
</Note>

## Keybind

The default keybind to open the menu is **F11**. It is automatically registered in GTA V as soon as the server starts.

```lua
Config.Keybind = 'F11'  -- enter any key, e.g. 'F9', 'F10', 'HOME'
Config.Keybind = false  -- disable keybind entirely
```

<Warning>
  The keybind is **only registered if `Config.UseRadial = false`** is set. As soon as the radial menu is enabled, `Config.Keybind` has no effect.
</Warning>

Players can change their personal binding at any time:

<Steps>
  <Step title="Open GTA V settings">
    In game, open the pause menu and navigate to **Settings → Keyboard / Controller**.
  </Step>
  <Step title="Find the resource">
    In the key bindings list, look for the resource name **`kt_extra_menu`**.
  </Step>
  <Step title="Rebind the key">
    Select the desired action and assign a new key. The custom binding is stored client-side and overrides the server default.
  </Step>
</Steps>

## Full configuration – command & keybind

```lua
-- Chat command to open the menu
Config.Command = 'vehiclemenu'

-- Keybind (only active if Config.UseRadial = false)
-- Set to false to disable the keybind
Config.Keybind = 'F11'
```

## Open the menu – all methods

<Tabs>
  <Tab title="Command">
    Type the following command in chat while sitting in a vehicle:

    ```bash
    /vehiclemenu
    ```

    The command name can be customized in `config.lua` via `Config.Command`. The command is always available, regardless of whether the radial menu or keybind is active.
  </Tab>
  <Tab title="Keybind">
    Press **F11** (or the key set in `Config.Keybind`) to open the menu.

    ```lua
    Config.Keybind = 'F11'
    ```

    Players can individually rebind the key in the GTA V settings under the resource name **`kt_extra_menu`**. The keybind is only available when `Config.UseRadial = false`.
  </Tab>
  <Tab title="Radial Menu">
    If `Config.UseRadial = true` is set, an entry **"Vehicle Customization"** appears in the radial menu as soon as the player is in a vehicle and has the required permissions.

    ```lua
    Config.UseRadial = true   -- enable radial menu
    Config.OxRadial   = true  -- use ox_lib radial menu
    Config.OxRadial   = false -- use qb-radialmenu
    ```

    For more setup details, see the [radial menu configuration](/configuration/radial-menu) page.
  </Tab>
</Tabs>

## Requirements to open the menu

Regardless of the chosen method, the following requirements apply:

- The player must be **inside a vehicle**. Outside a vehicle, an error message appears (_"You are not in a vehicle!"_).
- If `Config.OnlyDriver = true` is set, the player must be **the driver** (seat -1) of the vehicle. Passengers receive the message _"You must be the driver!"_.

```lua
Config.OnlyDriver = true  -- only the driver may open the menu
Config.OnlyDriver = false -- all occupants may open the menu
```
