<Info>
  KT UI is the shared component and minigame library used by all other KT resources. Install it **before** any other KT script.
</Info>

## 1. Download

Download `kt_ui` from [keymaster](https://keymaster.fivem.net) and extract to `resources/[kt]/kt_ui`.

## 3. Configuration

```lua config.lua
Config = {}
Config.Theme        = 'dark'      -- 'dark' | 'light' | 'auto'
Config.AccentColor  = '#d83533'
Config.SoundEffects = true
```

### Convars

```cfg server.cfg
set kt_ui:theme "dark"
set kt_ui:accent "#d83533"
```

## 4. server.cfg

```cfg
ensure ox_lib
ensure kt_ui   # must be started before other kt_* resources
```

<Warning>
  Always start `kt_ui` **before** other KT scripts in `server.cfg`. Other resources auto-detect it on boot.
</Warning>
