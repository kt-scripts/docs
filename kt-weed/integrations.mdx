## Adding a custom integration

Duplicate `bridge/_template.lua` and implement the four exposed methods:

```lua
return {
  addItem    = function(src, name, count) ... end,
  removeItem = function(src, name, count) ... end,
  hasItem    = function(src, name, count) ... end,
  notify     = function(src, msg, type)   ... end,
}
```
