## `getConfig`

Returns the merged runtime config.

```lua
local cfg = exports.kt_weed:getConfig()
print(cfg.MaxPlantsPerPlayer)  -- 8
```

## `getPlantStages`

```lua
local stages = exports.kt_weed:getPlantStages()
-- { 'seedling', 'vegetative', 'flowering', 'ready' }
```
