KT Extra Menu supports two menu systems: the widely used **ox\_lib context menu** and the **Lation Scripts Modern UI** for a more appealing, modern look. Which system you use depends on the resources installed on your server and your design preferences. Both systems use the same configuration options for access restrictions and notifications – only the look and some UI-specific settings differ.

## Select menu system

<Tabs>
  <Tab title="ox_lib (default)">
    The **ox\_lib context menu** is the default option and does not require any additional installation, since `ox_lib` is already a mandatory dependency of KT Extra Menu.

    ```lua
    Config.MenuSystem = 'ox'
    ```

    Notifications are emitted via the built-in `lib.notify` system of ox\_lib. The position of notifications can be controlled via `Config.Notifications.position`:

    | Value | Screen position |
    | --- | --- |
    | `'top-right'` | Top right _(default)_ |
    | `'top-left'` | Top left |
    | `'bottom-right'` | Bottom right |
    | `'bottom-left'` | Bottom left |

    ```lua
    Config.Notifications = {
        duration = 3000,
        position = 'top-right'
    }
    ```
  </Tab>
  <Tab title="Lation Modern UI">
    The **Lation Modern UI** provides a visually enhanced user interface with its own header and subheader in the menu window.

    <Warning>
      Lation Modern UI requires the separate **modernui** resource by Lation Scripts. It must be installed and started in `server.cfg` before `kt_extra_menu` is loaded. Without this resource, `Config.MenuSystem = 'lation'` will cause errors.
    </Warning>

    ```lua
    Config.MenuSystem = 'lation'
    
    Config.LationUI = {
        header = 'Vehicle Customization',
        subheader = 'Customize your vehicle',
    }
    ```

    **Startup order in `server.cfg`:**

    ```cfg
    ensure ox_lib
    ensure modernui
    ensure kt_extra_menu
    ```

    <ParamField path="Config.LationUI.header" default="'Vehicle Customization'" type="string">
      Large title text displayed at the top of the Lation Modern UI menu window.
    </ParamField>

    <ParamField path="Config.LationUI.subheader" default="'Customize your vehicle'" type="string">
      Smaller subtitle below the main title in the menu window. Suitable for a short description or a server name.
    </ParamField>
  </Tab>
</Tabs>

## Notification settings

The following options control the look and duration of notifications. They apply to both menu systems, although the `position` option is only considered when `Config.MenuSystem = 'ox'`.

<ParamField path="Config.Notifications.duration" default="3000" type="number">
  Display duration of the notification in milliseconds. `3000` equals 3 seconds. Increase this value if players frequently miss messages.
</ParamField>

<ParamField path="Config.Notifications.position" default="'top-right'" type="string">
  Screen position of notifications. Only effective when `Config.MenuSystem = 'ox'`. Possible values: `'top-right'`, `'top-left'`, `'bottom-right'`, `'bottom-left'`.
</ParamField>

```lua
Config.Notifications = {
    duration = 3000,
    position = 'top-right'
}
```
