Installing KT Extra Menu is straightforward and takes only a few minutes. You extract the script into your `resources` folder, add it to `server.cfg`, select your framework in `config.lua` and restart the server. A running **ox\_lib** instance on your server is required.

## Requirements

<Note>
  **ox\_lib** must be installed and active on your server before you can set up KT Extra Menu. Download ox\_lib from [github.com/overextended/ox\_lib](https://github.com/overextended/ox_lib) and make sure it is started **before** `kt_extra_menu` in `server.cfg`.
</Note>

## Installation steps

<Steps>
  <Step title="Download and extract the script">
    Download the latest release of KT Extra Menu from the [Lation Scripts Discord](https://discord.gg/A6UgrEhc8G). Extract the archive into the `resources` folder of your FiveM server.

    After extraction, the folder structure should look like this:

    ```text
    resources/
    └── kt_extra_menu/
        ├── client/
        │   └── main.lua
        ├── server/
        │   ├── main.lua
        │   └── version.lua
        ├── locales/
        │   ├── de.lua
        │   └── en.lua
        ├── config.lua
        └── fxmanifest.lua
    ```

    <Warning>
      Make sure the folder name is exactly `kt_extra_menu` (no version number in the folder name), since the `ensure` entry in `server.cfg` expects this exact name.
    </Warning>
  </Step>
  <Step title="Add to server.cfg">
    Open your FiveM server's `server.cfg` and add the following `ensure` entries. **ox\_lib must be started before kt\_extra\_menu.**

    ```cfg
    ensure ox_lib
    ensure kt_extra_menu
    ```

    <Warning>
      If `ox_lib` is started **after** `kt_extra_menu`, runtime errors will occur because the script already accesses ox\_lib functions on startup. Check the order carefully.
    </Warning>
  </Step>
  <Step title="Open config.lua and select your framework">
    Open the file `resources/kt_extra_menu/config.lua` in a text editor of your choice. At a minimum, set `Config.Framework` to match the framework you're using:

    ```lua
    Config.Framework = 'esx'      -- 'esx' | 'qbcore' | 'qbox' | 'standalone'
    ```

    Other configuration options such as menu system, language, command names and job restrictions can also be adjusted directly in this file. Detailed information can be found in the **Configuration** section.
  </Step>
  <Step title="Restart the server">
    Fully restart your FiveM server (or run `restart kt_extra_menu` in the server console if the server is already running and you only changed the configuration).

    Then check the server console for error messages. On successful installation, a confirmation message from KT Extra Menu including the current version number will appear.
  </Step>
</Steps>

## Next steps

After a successful installation, we recommend going through the [Quickstart](/quickstart) to configure the script with your framework and test it directly in game.
