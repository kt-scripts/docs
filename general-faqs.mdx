## Compatibility

<AccordionGroup>
  <Accordion title="Which frameworks are supported?" icon="layer-group">
    All KT Scripts resources support **ESX**, **QBCore** and **Qbox** out of the box. A standalone fallback is available where it makes sense.
  </Accordion>

  <Accordion title="Do I need ox_lib?" icon="box">
    Most resources require `ox_lib` for menus, notifications and progress bars. We recommend it on every server.
  </Accordion>

  <Accordion title="What inventory systems work?" icon="box-archive">
    `ox_inventory`, `qb-inventory`, `qs-inventory`, and `esx_inventoryhud`. Custom inventories can be added via the integrations file.
  </Accordion>
</AccordionGroup>

## Licensing & support

<AccordionGroup>
  <Accordion title="Is the source code obfuscated?" icon="file-shield">
    Configs, locales, and integration files are **fully open**. Core logic is escrowed via FiveM's official escrow system.
  </Accordion>

  <Accordion title="Can I use the script on multiple servers?" icon="server">
    Each purchase covers **one CFX server key**. Multi-server licenses are available on request.
  </Accordion>

  <Accordion title="How do I get updates?" icon="arrow-rotate-right">
    Re-download from [keymaster](https://keymaster.fivem.net) anytime. We post all changes in the per-resource changelog.
  </Accordion>
</AccordionGroup>
