## Version 1.0.0

**Initial release** — first release of KT Extra Menu.

### ✨ New features

- **Multi-framework support** Full compatibility with **ESX**, **QBCore**, **QBox** and **Standalone**. The framework is configured once in `Config.Framework`.
- **Two menu systems: ox\_lib & Lation Modern UI** Choose between the ox\_lib context menu (`Config.MenuSystem = 'ox'`) and the optional Lation Scripts Modern UI (`Config.MenuSystem = 'lation'`). Both systems offer a clean, performant interface.
- **Radial menu integration** Seamless integration with the **ox\_lib Radial Menu** and **qb-radialmenu**. Switchable via `Config.UseRadial` and `Config.OxRadial`.
- **Custom livery names** Assign custom labels for liveries of individual vehicle models directly in the configuration. The model name must be specified in lowercase:
  ```lua
  Config.CustomLiveryNames = {
      ['police'] = {
          [0] = 'LSPD Black/White',
          [1] = 'LSPD Slicktop',
          [2] = 'Sheriff',
      },
      ['ambulance'] = {
          [0] = 'Los Santos Medical',
          [1] = 'Fire Department',
      },
  }
  ```
- **Job restrictions** Restrict access to the menu to specific jobs. Compatible with all supported frameworks. With `standalone`, job checks are skipped:
  ```lua
  Config.RequiredJob = { 'mechanic', 'police' }
  ```
- **Multilingual (German & English)** Language switchable via `Config.Language = 'de'` or `'en'`. All menu texts and notifications are fully translated.
- **Automatic version check** On server start, the latest version is automatically queried. If an update is available, a colored notice appears in the server console with the current and latest version numbers as well as a download link.
- **Lua exports for external scripts** The current script version can be retrieved via an export:
  ```lua
  local version = exports['kt_extra_menu']:GetVersion()
  print('KT Extra Menu Version: ' .. version)
  ```
- **Lightweight and optimized** The script runs resource-friendly and causes no noticeable performance hit. Extras are scanned automatically in slots 0–20.
