A running log of changes shipped across the KT Scripts suite. For per-script history, see the changelog page inside each resource.

<Update label="Week of June 22, 2026" description="2026-06-28">
  ### ✨ New features

  - **[KT UI](/kt-ui/changelog) 1.2.0 — CircleProgress minigame.** A new timing-based minigame is available in the KT UI component library. Use it for skill checks, lockpicks, or any interaction that needs a quick player input. See the [CircleProgress component](/kt-ui/components/minigames/circle-progress).
  - **[KT UI](/kt-ui/changelog) 1.2.0 — `kt_ui:toast` export.** Trigger branded toast notifications from any resource without depending on a separate notification script. See [KT UI exports](/kt-ui/exports/client).

  ### 🔧 Updates

  - **[KT Weed](/kt-weed/changelog) 1.1.0 — Qbox support.** KT Weed now runs natively on **Qbox** alongside ESX and QBCore. No source changes required — set the framework in `Config.Framework`. See the [KT Weed install guide](/kt-weed/install).
  - **[KT Weed](/kt-weed/changelog) 1.1.0 — New `kt_weed:onHarvest` server event.** Hook into harvest actions from your own resources to award XP, log activity, or trigger custom rewards. See [KT Weed server exports](/kt-weed/exports/server).
  - **[KT Weed](/kt-weed/changelog) 1.1.0 — Improved plant prop streaming.** Plants now stream in at a longer distance, reducing pop-in on larger fields.
</Update>
